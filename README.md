# SuaVue Demo

This repo contains an implementation of a Suave Server hosting a VueJS Application

## Prerequisites

This application was built using a Mac. 

- Mono
- Node
- Paket
- Suave
- VueJS

## Server 

### Install Dependencies

```bash
./build.command
```

### Start Application

```bash
./start.command
```

## Web App

### Install Dependencies

```bash
cd public
npm install
```
